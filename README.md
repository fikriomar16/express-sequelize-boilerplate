
#### Usage

- Install Dependencies
> *(npm/yarn/pnpm) install*

- Run 
> *(npm/yarn/pnpm) start*

- Run Testing Unit
> *(npm/yarn/pnpm) test*

- Run Testing Unit with Debug
> *(npm/yarn/pnpm) test:debug*

*_Development Mode_*

- Run (Development)
> *(npm/yarn/pnpm) dev*
